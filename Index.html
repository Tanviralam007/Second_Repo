Making a list of Git:
To create a new repository follow the follwing rules:-
1. git initialization (git init)
2. adding (git add .)
3. commiting (git commit -m "ajfiuer")
4. brunch setting (git brunch -M main)
5. origin adding (git remote add origin https://github.com/Tanviralam007/Second_Repo.git)
6. pushing the code to the repository (git push -u origin main)

How to add the existing code to the repository after writing codes:
1. adding (git add .)
2. commiting (git commit -m "nfurefn")
3. pushing the code to the repository
